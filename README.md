## Description

## Usage
1. Restore NuGet Packages
2. Compile (requires [netcoreapp 3.0](https://dotnet.microsoft.com/download/dotnet-core/3.0))
3. Run QuickWindows.exe

Use `CTRL+SHIFT+C` to activate the app.
Type to search
Select the process
